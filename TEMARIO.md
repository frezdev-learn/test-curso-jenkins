### Temario Completo de Jenkins

#### Nivel Básico: Fundamentos de Jenkins

**Objetivo**: Comprender qué es Jenkins, cómo instalarlo y cómo usarlo para tareas simples de automatización.

1. **Introducción a Jenkins**

- ¿Qué es Jenkins y para qué sirve?
- Historia: De Hudson a Jenkins.
- Conceptos clave: Integración Continua (CI) y Entrega Continua (CD).
- Ventajas de usar Jenkins en el desarrollo de software.

2. **Instalación y Configuración Inicial**

- Requisitos del sistema (Java, memoria, etc.).
- Instalación en Windows, Linux y macOS - Uso de Docker para instalar Jenkins. - Acceso a la interfaz web y configuración inicial (desbloqueo del administrador).

3. **Interfaz de Usuario y Navegación**

- Vista general del tablero (Dashboard).
- Menús y opciones principales.
- Personalización básica de la interfaz.

4. **Creación de Proyectos Básicos**

- ¿Qué es un "Job" en Jenkins?
- Configuración de un proyecto freestyle.
- Conexión con un repositorio Git básico.
- Ejecución de un script simple (ejemplo: "Hello World").

5. **Gestión de Builds**

- ¿Qué es un build y cómo funciona?
- Visualización del historial de builds.
- Configuración de disparadores (triggers) manuales y automáticos (polling SCM).

6. **Introducción a Plugins**

- ¿Qué son los plugins y cómo instalarlos?
- Plugins esenciales: Git Plugin, Maven Integration, Email Extension.
- Actualización y gestión de plugins.
